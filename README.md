# chasseur-stellair

# --CONTRÔLE--
# Mouvement: Joueur contrôle le déplacement de son vaisseau avec la souris (délai).
# Attaque: Tenir Bouton-1 pour tirer le fusil.

# --OBJECTIF--
# Le joueur doit tuer le plus grand nombre d'ovnis et possibles pour obtenir des points de score.
# Les ovnis viennent en vagues qui durent 10 secondes chaque.
# Les asteroides/mines peuvent être détruits pour des points supplémentaires.
# On peut récupérer des ressources qui donnent des avantages au joueurs.
# À la 4e vague, le joueur doit détruire le boss du niveau pour continuer.
# En tuant le boss, le joueur gagne une récompense aléatoire et ses point de vie maximum augmentent.

# --MENU GRIS--
# Le joueur doit cliquer sur le bouton "Piloter" pour commencer le jeu.
# Dans le menu à droit, clique sur rejouer pour recommencer le jeu OU "Afficher les Scores" pour afficher les anciennes scores.
# Le menu montre les attributs du jeu, comme les vies restantes, niveaux, vagues, taux d'apparitions des ovnis, etc.

# --MENU ROUGE DU JOUEUR--
# Le menu rouge montre les attributs du vaisseau.

# -- BOSS DISPONIBLES --
# DoubleCannon: Le premier boss, il tire avec ses doubles cannons et peut seulment bouger horizontalement.
# Laser: Le deuxième boss, il tire un laser rapide devant. Lors du tire, le boss ne peut pas bouger. SON LASER EST DANGEREUX!
# Fonceur: Le troisième boss, un boss ultra aggresif. Il possède un triple cannon et peut bouger verticalement vers le joueur. Ses cannons prennent plusieurs secondes pour recharger, prenner avantage!
# NOTE: Les bosses alternent un après l'autre.

# -- ARMES DISPLONIBLES --
# DoubleCannon: Deux cannons qui tirent directement devant le joueur.
# Shotgun: Triple cannons qui tirent vers les côtés et le devant.

# -- RESSOURCES DISPONIBLES -- 
# Bouclier: Cercles blues donnent un bouclier au joueur. Un bouclier résiste une instance de dommage.
