# chasseur-stellair

# --CONTRÔLE--
# Mouvement: Joueur contrôle le déplacement de son vaisseau avec la souris (délai).
# Attaque: Tenir Bouton-1 pour tirer le fusil.

# --OBJECTIF--
# Le joueur doit tuer le plus grand nombre d'ovnis possible pour obtenir des points de score.
# Les ovnis viennent en vagues qui durent 10 secondes chaque.
# Les asteroides peuvent être détruits pour des points supplémentaires.
# On peut récupérer des ressources qui donnent des avantages au joueurs.
# À la 4e vague, le joueur doit détruire le boss aléatoire du niveau pour continuer.
# En tuant le boss, le joueur gagne une récompense aléatoire.

# --MENU--
# Le joueur doit cliquer sur le bouton "Piloter" pour commencer le jeu.
# Dans le menu à droit, clique sur rejouer pour recommencer le jeu OU "Afficher les Scores" pour afficher les anciennes scores.
# Le menu montre les attributs du jeu, comme les vies restantes, niveaux, vagues, taux d'apparitions des ovnis, etc.

# --MENU D'ATTRIBUTS DU JOUEUR--
# Le menu rouge montre les attributs du vaisseau.
